TekWurx uControl is supplied in 2 formats

  • As a virtual appliance running the Ubuntu OS, version 12.04.2 LTS (Precise) running the MariaDB.
  • As a .War file install that allow you to install on to your physical or virtual server running a supported Linux distribution
