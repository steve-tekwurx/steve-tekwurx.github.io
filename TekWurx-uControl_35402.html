uControl Documentation 5.0 : TekWurx uControl

Created by Steve Hicks on Jul 08, 2016


TekWurx uControl puts YOU in control, to substantially reduce the cost and complexity of BMC ADDM ownership.

By cutting out the need to learn complex reporting and scripting languages, uControl puts application mapping in your hands, producing detailed reports on which applications sit where and how they work together.

Further, uControl can keep you clearly and simply informed of the changes in the application topography and enable you to easily manage this change.

BMC ADDM combined with uControl enables you to:

  • Improve IT efficiency and reduce costs
  • Quickly create Application maps
  • Proactively monitor drift in application maps
  • Manage your entire discovery solution from a single pane of glass

uControl Features

uMap

Automated Application Mapping

uMap™ - A new enterprise software solution that forms part of the uControl™ suite provides end-users with the ability to map Application / Service models through an intuitive UI Wizard.

You don't need a programmer or a technical consultant, you don't even need to know TPL or any other scripting language. Everything can be accomplished quickly and easily using a point and click interface. 

uMap™ :

  • Automated Application and Service Modelling
  • Reduces effort by over 80%
  • Reduces reliance on external expertise
  • Reduces costs associated with Application Modeling

 

uChange

Application Lifecycle Monitor

After modeling your Business Applications through ADDM the next challenge is understanding Application change and managing Application model updates back within ADDM

uChange™ - A new enterprise software solution that forms part of the uControl™ suite allows users to understand changes to modelled applications and control any updates back within ADDM. Through an intuitive user interface, businesses can clearly see and manage any alterations to the application topography

uChange™ :

  • Automated Application lifecycle analysis and updates
  • Increased Productivity
  • Reduced ADDDM management Costs

uManage

ADDM Appliance Automation and Management

uManage™ - A new enterprise software solution that forms part of the uControl™ suite provides end-users with improved control over the management and delivery of BMC ADDM.

An intuitive module that reduces the skill level required to support BMC ADDM within your organisation by allowing your team to use a wizard type interface and single pane of glass to manage your discovery deployment.

uManage™ :

  • Centralized management of all ADDM appliances
  • Increased Productivity
  • Reduced ADDM management Costs

uReport

Centralised Reporting

uReport™ - A new enterprise software solution that forms part of the uControl™ suite enables users to analyse reports without the need to learn or use TQL language. Through the friendly UI, uReport™ quickly shows users what they need to know about their appliances and application architecture.

 

uReport™ :

  • Centralized from all ADDM appliances
  • Increased Productivity
  • Reduced reporting times.

 

 

uPolicy

Centralised Policy Dashboard

uPolicy™ - A new enterprise software solution that forms part of the uControl™ suite enables users to analyse policy based reports and quickly identify breaks, export the results and forward on to a team to quickly resolve the issues

 

uPolicy™ :

  • Centralized from all ADDM appliances
  • Single Dashboard view with alerting to quickly identify issues
  • View in uControl or export to Microsoft Excel
