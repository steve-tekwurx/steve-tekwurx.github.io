uControl Documentation 5.0 : uMap Tools

Created by Steve Hicks on Jul 08, 2016

uMap tolls provide the ability to build Software from Processes , Filesystems or Services.

Building Software From Processes

  1. Navigate to uMap -> uMap Tools -> Build From Processes
  2. Add matching criteria into the provided form to filter the results in the table.
  3. Once the table only shows the results you wish to model as a single software instance type click next
  4. Provide a Software Type (eg. Oracle Database Server)
  5. If required, you can provide regex to match a version string from either the command or arguments field of the process
  6. Submit
