uControl Documentation 5.0 : Managing Excludes

Created by Steve Hicks on Jul 08, 2016

uManage allows you to upload a list of IPs and assign the exclude list to 1 or more selected appliances

Define An Exclude Range

  1. Navigate to uManage -> Discovery -> Excludes -> Define
  2. Select 'Define Exclude' and enter a name and the IP range to exclude (Comma separated string)
  3. Select 'Add Range' to complete the definition of the exclude range.

Apply the Exclude Range

  1. Navigate to uManage -> Discovery -> Excludes -> Schedule
  2. Select 'Define Schedule'
  3. Enter a Name, description and select 1 or more ADDM appliances to deploy the exclude range to
  4. Finally, click 'Schedule' to deploy the exclude range to the ADDM appliance(s)
