uControl Documentation 5.0 : Managing TKUs

Created by Steve Hicks on Jul 08, 2016

uManage allows you to upload A TKU package and schedule the deployment to one or more appliances

Ensure that the default packet size on the underlying database is configured large enough for the TKU package.

For MySQL / MariaDB this can be completed by setting the following in the /etc/my.cnf file:

[mysqld]
max_allowed_packet = 1073741824

 

Upload TKU Package

  1. Navigate to uManage -> Discovery -> TKU -> Upload
  2. Select 'Choose File' and select the TKU package from you desktop machine and select 'Upload TKU Package'

Schedule the TKU Deployment

  1. Navigate to uManage -> Discovery -> TKU -> Deploy
  2. Select the TKU package from the 'Files Available To Upload' table
  3. Enter the schedule and select the appliances to deploy to and click save
