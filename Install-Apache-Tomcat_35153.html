uControl Documentation 5.0 : Install Apache Tomcat

Created by Steve Hicks on Jul 08, 2016

uControl is a dynamic Java application running under Apache Tomcat. To install Apache Tomcat:

  1. Download the Core Apache Tomcat zip package from http://tomcat.apache.org/ 
  2. unZip under the uControl users home directory
  3. Navigate to the <uControl_homeDir>/Apache*/webapps directory
  4. Create a uControl directory with standard permissions (755)
