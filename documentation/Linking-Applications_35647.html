uControl Documentation 5.0 : Linking Applications

Created by Steve Hicks on Jul 08, 2016

BMC ADDM will use discovered communications to infer relationships and dependencies between software components and applications. uControl also enables you to set dependencies between Applications.

Application Dependency Details enables you to link this application to other applications. When the final application pattern is submitted to ADDM and discovery is completed – relationships will be created between the applications showing a dependency exists.

    1. Click Add application



    2. uMap displays all the applications imported into uMap whether or not these have been modelled yet. If they have not been modelled uMap will store the relationship and create the dependency when the other application is created.



    3. When creating the application link you can also set whether this is a critical dependency and if so you can include a description.
