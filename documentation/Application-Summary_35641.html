uControl Documentation 5.0 : Application Summary

Created by Steve Hicks on Jul 08, 2016

 

On the completion of all modelling steps a summary of the application including:

  • Hosts
  • Unmatched Hosts
  • Software Products
  • Software Processes
  • Application Dependencies

To successfully deploy to ADDM, the unmatched hosts should be zero. If this is not the case, on submit the application will be flagged for admin review.

 

Click on the 'Submit to appliance' button to complete the modelling process for the application
