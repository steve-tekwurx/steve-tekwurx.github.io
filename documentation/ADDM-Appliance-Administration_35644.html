uControl Documentation 5.0 : ADDM Appliance Administration

Created by Steve Hicks on Jul 08, 2016

To view current status of ADDM appliances:

  1. Login to uControl with a user account that has permissions to use uManage
  2. Click the uManage icon



  3. The Configuration TAB will display all the ADDM appliances that you have integrated with uControl




  4. For each ADDM appliance uControl will show:
    1. Last reboot time
    2. Whether the UI is available to users
    3. IP address of Appliance
    4. Version of ADDM on the appliance
    5. TKU version installed on ADDM
    6. Diskspace
    7. Datastore Size
    8. Current discovery mode (playback, record or None)
    9. Any errors encountered with discovery
  5. The reboot button enables you to reboot the ADDM appliance host
  6. The restart button enables you to restart the ADDM services
  7. The start button enables you to start the ADDM services
  8. The stop button enables you to stop the ADDM services
  9. The ECA Error count button enables you to identify and see the current ECA errors from pattern on the ADDM appliance
  10. The Launch UI button enables you to access the ADDM UI.
  11. Stop Discovery provides the capability to stop (and start) discovery on the ADDM appliance. When discovery is running the button is green and read "Stop Discovery". If discovery is stopped the button is red and reads "Start Discovery".
  12. Start Maintenance Mode enables you to place an appliance into Maintenance mode - preventing users from accessing the ADDM UI. If you place an appliance into maintenance mode the button turns red and enables you to disable maintenance mode.
  13. The service status section of the page will show the current state of the ADDM services
  14. The baseline status section will show the current baseline status of the addm appliance
  15. The Show Cluster Information' will show cluster information of the appliance

Alerts

uManage generates alerts allowing you to monitor, and be alerted to, issues with your ADDM appliances.

The icons within the ADDM Status section of the page have green ticks set when everything is healthy. The left hand gif provides information about ADDM itself and generates alerts when either the Appliance itself becomes unavailable or the UI becomes unavailable. The right hand gif provides information about the disk space on the appliance and generates alerts when utilisation goes over a pre-defined limit.

Underneath the Appliance icons is an error count button. This shows any errors detected on the appliance during the last discovery run. To view these errors click the button and a pop-up will display the error message and cause of the error.
