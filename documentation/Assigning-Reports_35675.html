uControl Documentation 5.0 : Assigning Reports

Created by Steve Hicks on Jul 08, 2016

uControl allows the assignment of report to other users / email addresses by a uReprot + Admin user.

When assigning reports you can either just assign (Report available under the My Reports of the user) or Assign and schedule (As before + email reports to user)

Assigning Reports

  1. Login as an Admin user
  2. Navigate to uAdmin -> uReport -> Assign Reports
  3. Select the report you would like to assign
  4. Either enter an email address / address list (separated by ';') or select a user from the table provided
  5. Configure the options - either assign or assign and schedule
  6. Click 'Link / Assign'
