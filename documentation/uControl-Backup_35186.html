uControl Documentation 5.0 : uControl Backup

Created by Steve Hicks on Jul 08, 2016

The uControl application has a built in backup utility that will:

  • Create a snapshot of the underlining DB
  • A copy of the configuration setting for uControl

Configure uControl Backup

  1. Navigate to uAdmin -> System -> Backup

  2. Select the time and days of the week to completed the uControl backup.
