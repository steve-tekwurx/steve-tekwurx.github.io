uControl Documentation 5.0 : Installation

Created by Steve Hicks on Jul 08, 2016

New installations of Tekwurx uControl are installed by package install files or a downloaded virtual machine. See 'Installing a uControl VM Appliance' for more information on installing the Virtual Appliance or 'Installing uControl Software' for installing uControl on to pre-existing hardware and OS.

It is recommended that production instances of uControl are installed using the packages provided onto a provided Linux built server
