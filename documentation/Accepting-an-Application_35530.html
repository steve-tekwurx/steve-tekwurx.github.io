uControl Documentation 5.0 : Accepting an Application

Created by Steve Hicks on Jul 08, 2016

When an application owner accesses uMap the My Models table will show the current status of Applications that they have been assigned. These will fall into 2 categories:

  • New: Applications that have been assigned to the application owner but not accepted
  • In Progress: Applications that have been accepted by the application owner

The first task is to either accept or decline the application.

 

To Accept the Application

  1. If you are the correct application owner for the assigned application and can provide the details in order to model the application then you should tick the check box for the application and then choose “Accept Selected”.
    1. The application will move to the In Progress TAB.
  2. If you are not the correct application owner for the assigned application then you should select “Decline Selected”.
    1. This will remove the application from your view and notify the uMap administrator that they need to re-assign the application to a different user.
