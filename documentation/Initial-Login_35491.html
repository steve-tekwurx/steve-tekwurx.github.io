uControl Documentation 5.0 : Initial Login

Created by Steve Hicks on Jul 08, 2016

Logging in to uControl

  1. Enter http://ip_address/uControl into your browser. (Where ip_address is the address of your uControl appliance).



  2. On the login page enter your username and password and click login.
  3. The main uControl dashboard screen will be displayed



  4. The dashboard view shows 2 overview charts for uMap and uChange.
    1. The uMap chart shows the number of applications that have been assigned to the user account you have logged in with and whether they are Awaiting Acceptance, Being Built or Signed Off.
    2. The uChange chart shows the number of change alerts seen by comparing the Application Gold Build against the latest scan from BMC ADDM for applications owned by the account you have logged in as.
  5. If the user account has any new applications assigned to work on which are currently Awaiting Acceptance then the menu icon for uMap will display a red icon with an indication of how many new applications have been assigned. 



  6. To access the new applications click the uMap icon.
