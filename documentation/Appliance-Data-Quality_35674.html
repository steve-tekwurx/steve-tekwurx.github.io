uControl Documentation 5.0 : Appliance Data Quality

Created by Steve Hicks on Jul 08, 2016

The Appliance Data Quality tool under uManage is use to identify data differences of hosts and software between appliances configured under uManage

Appliance Data Quality

  1. Navigate to uManage -> Tools -> Appliance Data Quality



  2. Select the source and destination appliances



  3. Hit the 'Run Validator' button and view results for Host and Software differences
