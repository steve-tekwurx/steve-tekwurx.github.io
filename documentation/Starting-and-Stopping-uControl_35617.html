uControl Documentation 5.0 : Starting and Stopping uControl

Created by Steve Hicks on Jul 08, 2016

Starting uControl

From the <uControl_homeDir>/Apache*/bin directory run the startup.sh script

Stopping uControl

From the <uControl_homeDir>/Apache*/bin directory run the shutdown.sh script

 

Check if uControl is running or not

To check if the uControl application is running run the following command on the CLI:

uControl Application Process is Running
ucontrol@tekwurx:~/apache-tomcat-8.0.15/bin$ ps -ef | grep -v grep | grep tom

ucontrol 17149     1  1 Jan23 ?        02:52:24 /usr/bin/java -Djava.util.logging.config.file=/home/ucontrol/apache-tomcat-8.0.15/conf/logging.properties -Djava.util.logging.manager=org.apache.juli.ClassLoaderLogManager -Djava.endorsed.dirs=/home/ucontrol/apache-tomcat-8.0.15/endorsed -classpath /home/ucontrol/apache-tomcat-8.0.15/bin/bootstrap.jar:/home/ucontrol/apache-tomcat-8.0.15/bin/tomcat-juli.jar -Dcatalina.base=/home/ucontrol/apache-tomcat-8.0.15 -Dcatalina.home=/home/ucontrol/apache-tomcat-8.0.15 -Djava.io.tmpdir=/home/ucontrol/apache-tomcat-8.0.15/temp org.apache.catalina.startup.Bootstrap start
uControl Application Process is Stopped
ucontrol@tekwurx:~/apache-tomcat-8.0.15/bin$ ps -ef | grep -v grep | grep tom
