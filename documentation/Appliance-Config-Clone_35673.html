uControl Documentation 5.0 : Appliance Config Clone

Created by Steve Hicks on Jul 08, 2016

The Appliance Config Clone tool under uManage is use to replicate ADDM configuration between appliances configured under uManage

 

The following configurations can be replicated:

  • BaseLine Configuration
  • Custom Reports
  • Custom Visulisations
  • Custom Dashboards
  • ADDM Credentials
  • Custom Taxonomy

 

Appliance Config Clone

  1. Navigate to uManage -> Tools -> Appliance Config Clone



  2. Select the source and destination appliances



  3. Select the options to clone from source to destination and press 'Clone'
