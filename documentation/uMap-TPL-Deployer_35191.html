uControl Documentation 5.0 : uMap TPL Deployer

Created by Steve Hicks on Jul 08, 2016

Once application modes are defined in uControl they are deployed to ADDM on schedule.

Configure uMap TPL Deploy Schedule

  1. Navigate to uAdmin -> uMap -> Manage uMaps
  2. Under the 'uMap Production Deployment Configuration' set the time and date for the production TPL deployer to run.


Be sure to scheduled the TPL deployer at a time the ADDM appliances defined in the uControl role is idle / not scanning as the TPL upload will have a performance impact on the ADDM system.

The TPL test deployer will trigger on the submit of any completed uMap application by any user
