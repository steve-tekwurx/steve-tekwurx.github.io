uControl Documentation 5.0 : Required Packages

Created by Steve Hicks on Jul 08, 2016

The following packages are required to be installed prior to installing uControl:

  1. Sun Java JDK version 7 or above 
  2. MariaDB version 5.5 
  3. mysql-workbench
  4. zip
  5. apache 2 with mod_proxy (mod_proxy_http, mod_proxy_ajp)
  6. ssh server

 

While Sun Java version 7 is supported the preferred version is currently 8

MariaDB is preferred for larger deployments of uControl although MySQL is also supported
