Created by Steve Hicks, last modified on Jul 08, 2016

Licensing Entitlement

Tekwurx uControl uses a custom built license method to license it's modules for use. To license your instance of uControl raise a support issues using the appropriate channels.

Licensed Modules

uMap - Based on the total number of imported applications to model

uManage - Based on the number of ADDM appliances configured
