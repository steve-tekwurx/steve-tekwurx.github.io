uControl Documentation 5.0 : Define Applications within uMap

Created by Steve Hicks on Jul 08, 2016

Application can be added to uControl in 2 ways:

Adding Applications Via CSV

  1. Login to uControl as the administration user
  2. Click the uMap icon



  3. Select the Manage Application drop down menu item and select 'Import/Create'

  4. Under 'Import Application Via CSV' select the 'Find File to upload' link and select the CSV file that contains your applications

    CSV File Format

     

    The CSV file requires 5 fields in the following order:

    ID - The application ID

    Name - The Application Name

    Description - The Application description

    Owner Email - The owners associated to the application (Email Addresses)

    Linked Hosts - Hosts that are part of the application

     

    Each field should be comma seperated. For fields that required multiple entries, wrap in double quotes. eg:

    12394,Test Import, Test CSV Import application,"steve@tekwurx.com","lon2p,lonp1150,stevetest"

    A header row is also required although it is not used.

  5. Once the file has been selected click on the 'Import Application' button to import the application to uMap.

Adding Applications Manually

  1. Login to uControl as the administration user
  2. Click the uMap icon



  3. Select the Manage Application drop down menu item and select 'Import/Create'
  4. Under the 'Delete, Create and Manage' section select the 'Create New' button.
  5. Provide a Name, Application ID and Description and click 'Create'
