uControl Documentation 5.0 : Providing Application Details

Created by Steve Hicks on Jul 08, 2016

Each application that you have accepted will appear on the In Progress tab. 

  1. Select the In Progress TAB to see the applications and their current status.



  2. To work on an application click the application row in the table – this will open up the screens where you select the application details.



  3. The first step is to confirm the details about the application, namely the description, is correct.
  4. If you need to add other users to assist in the completion of the application information then click Add additional owner. Whoever you assign to the application will be notified by email and they will be able to enter information in parallel. uControl will hold the application in the information gathering stage until all assigned parties have completed their tasks.
  5. If all details are correct press “Next Step’ otherwise amend the description, update and then click next step.
