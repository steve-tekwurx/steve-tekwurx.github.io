uControl Documentation 5.0 : Installing uControl

Created by Steve Hicks on Jul 08, 2016

Packaging

The Virtual Appliances are packaged as a compressed zip file that contains either a

  • VMware virtual machine
  • VirtualBox virtual machine

Specifications

The specification is based on the hosts imported from ADDM. Use the following as the minimum requirements

Number Of HostsCPURAMDisk
Up to 80001 (2 vCPU)8GB100GB
8000 - 16,0001 (2 vCPU)12GB100GB
16,000 - 24,0001 (2 vCPU)16GB150GB
24,000 - 32,0001 (2 vCPU)20GB200GB
32,000 - 40,0001 (2 vCPU)24GB200GB
  • Network Interface Card: 1 configured with static IP

 

For deployments above 40,000 hosts please contact TekWurx Development Team for guidelines.

 

 

Installation

The uControl software can be installed in 2 ways. Either by packages or by installing a pre built VM from TekWurx

Only servers can be used in a production environment. Workstations and laptops are not production specification.
