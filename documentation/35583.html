uControl Documentation 5.0 : uMap / uChange Linked Hosts and Alerts

Created by Steve Hicks on Jul 08, 2016

The linked hosts and alerts job return and alert (For signed off application) the hosts that / software that has been built in the ADDM appliance.

Configure uMap and uChange linked Hosts and Alerts schedule

  1. Navigate to uAdmin -> uMap -> Linked Hosts and Alerts
  2. Under 'Configuration' set the time and date for the schedule to run.
