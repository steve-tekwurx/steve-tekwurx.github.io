Logging in to uControl

  1. Enter the following URL into your browser (where IP address is the address of your virtual machine).
    1. http://<IP_ADDRESS>:8080/uControl
  2. Log in as the admin user with the admin username and password. The initial values for uControl are:
    1. Username: admin
    2. Password: pleasechangeme
  3. Click login
  4. Change the admin password as follows:
    1. Click the 'Manage Account' link at the top right hand side of the UI

    2. In the pop-up box enter your new password and click update
