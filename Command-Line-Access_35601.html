Logging in to Command Line

Some features of uControl need command line access to configure. For example setting the static IP address, NTP settings or starting and stopping services.

Default command line passwords

uControl is configured with the following default command line user accounts and passwords:

  • ucontrol â€“ ucontrol

You should log in as the ucontrol user and change this password immediately.

To log in either use the VMware Server Console or SSH in to the appliance using a terminal such as PUTTY.

To change the ucontrol password

  • As the ucontrol user use the passwd command the change the password.
  • Enter a new password when prompted.
