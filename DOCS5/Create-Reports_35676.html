uControl Documentation 5.0 : Create Reports

Created by Steve Hicks on Jul 08, 2016

If you are configured as a standard uReport user you have the ability to create a report to run and schedule against ADDM

Defining a Report

To define a report navigate to uReport -> My Reports

  1. Select 'Create Report'
  2. Give the report a name and description and enter the ADDM search query to execute
  3. Click OK to save the report.
  4. The Report will now be defined under My Reports. This report will now be updated based on the default schedule / frequency.
