uControl Documentation 5.0 : Managing Scans

Created by Steve Hicks on Jul 08, 2016

uManage allows you to import - on a regular basis a list of subnets from a remote ssh location. Once the import is completed uControl will understand what is new / existing or missing compared to the last import. It will then, on schedule, run these subnets through a 'Sweep Scan' in ADDM and retrieve the responsive IPS. These IP's are then configured to run against production ADDM scanners to completed a full discovery of the assets. uManage Scans ensures the ADDM production scanners only try to scan IPs that are valid and responsive in the discovery estate, minimising your production scan windows and ensuring through the uManage Scans Rules Engine that nothing is missed within the configured / supplied subnets.

Import Subnet information

The import of this information is configured through the uAdmin -> uManage section of uControl. Further information of this can be found here.

Results Post Import

On completion of the file import the uManage -> Discovery -> Scans -> Define page will be populated with the following information:

  • Scannable networks not linked to a schedule
  • Scannable networks not in last import
  • Scannable networks NEW in last import
  • List of Scannable networks

If emails are enabled and an email address was configured for email updates post import, the above information will be supplied within the email that is sent

 

Creating Schedules

Schedules are created per appliance. Appliances that are available to be selected are based on the uAdmin -> uManage -> "Manage Appliance Roles" definitions.

To create a new schedule:

  1. Navigate to uManage -> Discovery -> Scans -> Scanner and drop down the appliance that you would like to create a schedule for.
  2. Select 'New Schedule'
  3. Input a Name, Description and configure the time / dates schedule to complete the run.
  4. Optional - Add subnet IP ranges with the 'Add Network' section (This can be don from the Define tab from a multi select table)
  5. Create Schedule

Sweep Scan - this option is to control the type of scan. This has 2 options:

Linked - Allow the adding of subnet definitions to the scan

All Linked - For the appliance selected take all the Linked subnets and create a 'Master' scan. Good for weekly / monthly scans.

 

Edit Schedules

To edit a schedule navigate to uManage -> Discovery -> Scans -> Schedules.

Select the schedule you would like to edit. This will open up the schedule definition and allow editing.

 

View Status and Messages

To view the status or any message from the uManage Scans navigate to uManage -> Discovery -> Scans -> View Log
