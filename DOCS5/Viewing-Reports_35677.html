uControl Documentation 5.0 : Viewing Reports

Created by Steve Hicks on Jul 08, 2016

uControl contains some in built reports the are viewable from the uReport Dashboard.

Dashboard Reports

Reports are updated based on the uAdmin -> uReport schedules. A uControl administrator will need to enable and configure each one as required. If no results are returned either there is no data that meets the criteria in ADDM or the schedule / report has not been enabled.

Â 

To view a report click on the name. A screen is shown with the results and the following actions are available:

  • Schedule - Configure this report to be schedule and results sent to the logged in user
  • Download as CSV - download the results manually as a CSV file.

  • Close - Return to the Dashboard

    Ensure that when scheduling uControl has been configured with en email system.

All scheduled reports are viewable under My Reports
