uControl Documentation 5.0 : Login Checker

Created by Steve Hicks on Jul 08, 2016

The Login Checker tool under uManage is uses to test ADDM discovery access to hosts using the ADDM configured credentials / Windows proxies

Login Checker

  1. Navigate to uManage -> Tools -> Login Checker



  2. Select 'New Test', supply ip address details and click submit



  3. Once the test is complete, the results will be shown in the table
