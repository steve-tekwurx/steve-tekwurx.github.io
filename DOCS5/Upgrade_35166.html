Upgrading

The following sections describe the upgrade paths from previous releases of the TekWurx uControl software to the latest GA

Create a backup

Before completing any steps in this section it is highly recommended that a data export and a backup of the original deployed uControl application is completed.

 

 

Before starting the upgrade process, you will need

  • The uControl (uControl.war) package

These required files / packages will be available under your support login to the TekWurx Support site.

If installed using the default path, the UCONTROL_INSTALL_DIR would be /uControl/tomcat8

 

  1. Stop the uControl Application

    <UCONTROL_INSTALL_DIR>/bin/shutdown.sh

     

  2. Remove the existing version of the uControl application

    rm -rf <UCONTROL_INSTALL_DIR>/webapps/uControl

     

  3. Unpack the uControl.war file

    cd <UCONTROL_INSTALL_DIR>/webapps
mkdir uControl
cd uControl
unzip uControl.war

     

  4. Start the application

    <UCONTROL_INSTALL_DIR>/bin/startup.sh
