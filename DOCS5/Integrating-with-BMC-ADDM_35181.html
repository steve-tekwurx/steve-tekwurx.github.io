uControl Documentation 5.0 : Integrating with BMC ADDM

Created by Steve Hicks on Jul 08, 2016

After having installed uControl one of the first tasks is to integrate uControl with your ADDM appliances.

In oder to integrate with ADDM you will need the following information:

  • URL of ADDM Applicance
  • IP Address of ADDM appliance
  • SSH user name
  • SSH user password
  • ADDM System user
  • ADDM System password

 

Adding ADDM appliance to uControl

  1. Login to uControl as the administration user
  2. Click the uAdmin icon



  3. On the administration page select the uManage TAB





  4. On the uManage TAB under Manage Appliances TAB click Create Appliance.



  5. On the pop-up enter the information required to connect uControl to ADDM. The working DIR is the location on the remote ADDM appliance where you would like uControl to copy and upload files to. It is recommended that the working DIR is the /tmp DIR. Once completed, click OK. If you are using HTTPS UI access with ADDM then set SSL to yes.

Assigning roles to ADDM appliances


Once ADDM appliances have been added to uControl they can be assigned roles which controls how uControl interacts with the Appliance. For example if you have a DEV appliance which you use for testing TPL patterns before you deploy to a production appliance then you can assign a DEV TPL appliance and a PROD TPL appliance in uControl. This will result in all TPL being deployed to your DEV appliance and once tested deployed to your PROD appliances.

  1. Login to uControl as the Admin user and select the uAdmin module
  2. On the uManage TAB screen click the Roles TAB.



  3. Depending on the role that you wish uControl to use the specific ADDM appliance for - click the Add Appliance button and select the required ADDM appliance from the list.
