uControl Documentation 5.0 : uManage Subnet IP File Import

Created by Steve Hicks on Jul 08, 2016

To import subnets for use in uManage scanning the following job needs to be scheduled.

Configure uManage Subnet File Import

  1. Navigate to uAdmin -> uManage -> Manage Subnet Scanning

  2. Under'Import Configuration' set the information for the file import location, connection details and layout.

     

    The layout column references are a zero based index. eg, the first column in the import is referred to at '0'
