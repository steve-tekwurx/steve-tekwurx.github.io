uControl Documentation 5.0 : uMap Visualisations

Created by Steve Hicks on Jul 08, 2016

Visualisations in uControl allow you to visualise your applications at each part of the modelling workflow

Visualisations

Visualisations are available from the following screens:

  • uMap modelling steps
  • My Applications -> To Validate
  • My Applications -> Signed Off
  • Manage Applications -> Signed Off

 

These are 2 menus available in each visualisations

 

Tools

  • Edit
  • Print
  • Save

 

 

Contextual Overlays

  • Environmental - Zone
  • Environmental - Region
  • Network - Network Devices

 

The ADDM overlays for uMap Visualisations are controlled / defined by the uManage appliance setup attributes that are set on the appliances.
