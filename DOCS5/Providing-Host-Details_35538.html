uControl Documentation 5.0 : Providing Host Details

Created by Steve Hicks on Jul 08, 2016

Step 2 is about selecting the hosts that are part of your application.

If uControl has been integrated with a CMDB or existing system of record which contained details about your application then the host list may be pre-populated. The table entitled "Servers available to be selected" contains all discovered hosts from ADDM. The table entitled "Servers identified as being used by this application" are the hosts that have been linked to the application

NOTE: If you do not know the hosts that belong to your application then you can skip this step by clicking the next button.

 

  1. Confirm that the hosts in the list are correct. If they are not then either add or remove hosts as required.



  2. If you need to add Hosts then you can either search by hostname or by IP address. From this table you can filter for the hosts that are part of your application and add them to your application record in uMap by clicking the appropriate row.
    1. If you wish to add hosts to the application that have not yet been discovered by BMC ADDM then click the “Add Servers Manually” button and complete the details required.


  3. As you select hosts from the list the host entry row will disappear from the "Servers available to be linked" table and appear in the "Servers identified as being used by this application" table to indicate that you have already selected that host and added to your model. If you need to remove the host from the model then select the host row and click the Remove Selected Hosts button.
    1. If you should select a host that is part of a cluster then uControl will automatically add all parts of the cluster to the application.
  4. When you select a host from the list a pop-up will ask you to add the environment details for the host and then confirm that you wish to add the host to the application.



  5. You can also use the Remove Selected Hosts button to bulk remove hosts and the Add Servers Manually button to add hosts into the application build that have not currently been discovered by ADDM.
