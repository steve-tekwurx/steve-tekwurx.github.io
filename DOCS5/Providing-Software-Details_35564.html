uControl Documentation 5.0 : Providing Software Details

Created by Steve Hicks on Jul 08, 2016

Once you have completed linking software to the application be sure to check the information button at the top right of the screen. If you have linked software on a host not selected in the first step the host will be automatically added / linked to the application (X Extra Hosts). Be sure to revisit the Host screen and update the environment.

Products

Step 3 enables you to select the software components that comprise your application. From this step you can select all the software products, Filesystems, Packages and services that identify your application.

Linking Products

  1. The software list displayed by default shows all software that BMC ADDM has discovered with a unique identifier. This can be searched and filtered by clicking the magnifying glass icon at the top of the table and then typing your criteria. This will return all matching software for all scanned hosts discovered by ADDM. To make searching easier you can select "Products filtered by hosts linked to this application" which will only display Software Instances with unique identifiers running on the hosts you selected in the previous step.



  2. If the software does not cover all hosts in the list you have the ability to select these by running processes in step 4.
  3. When you select the Software Component that belongs to your application â€“ uControl enables you to select whether the dependency on that software is critical or not and provide a description for this. This information can be used within ADDM to define the relationships between components and also by the CMDB.

Linking Filesystems

  1. Select the filesystem button to list the available filesystems discovered by ADDM



  2. Use the filter boxes on the first row of the displayed grid to allow filtering the available filesystems. Select a single or select multiple rows and click the Add FS button to link them (and the hosts they are on) to the application.



  3. Provide a name to group the filesystem nodes together and selected OK.

Linking Packages

  1. Select the Packages button to list the available packages discovered by ADDM



  2. Use the filter boxes on the first row of the displayed grid to allow filtering the available packages. Select a single or select multiple rows and click the Add Package button to link them (and the hosts they are on) to the application.



  3. Provide a name to group the package nodes together and selected OK.

Linking Services

  1. Select the services button to list the available services discovered by ADDM

  2. Use the filter boxes on the first row of the displayed grid to allow filtering the available services. Select a single or select multiple rows and click the Add Service button to link them (and the hosts they are on) to the application.



  3. Provide a name to group the service nodes together and selected OK.
