uControl Documentation 5.0 : Operating System

Created by Steve Hicks on Jul 08, 2016

Operating Systems

The following Operating systems are supported to run uControl:

  1. Red Hat Linux 6 - 64Bit
  2. Ubuntu Server 12.04 - 64Bit
  3. Ubuntu Server 14.04 - 64Bit

Storage Requirements

The following is a recommended disk layout configuration for uControl:

MountUsed For
/usruControl Application Install Directory
/dbuControl Database files
/backupuControl DB backup files - 2nd Disk Preferred

Firewall Requirements

For RedHat linux server the following configuration in iptables is required

-A INPUT -m state --state NEW -m tcp -p tcp --dport 8080 -j ACCEPT
