uControl Documentation 5.0 : First Steps

Created by Steve Hicks on Jul 08, 2016

This section describes how you can start using uControl.

Logging in to uControl

  1. Enter http://ip_address/uControl into your browser. (Where ip_address is the address of your uControl appliance).



  2. On the login page enter your username and password and click login.
  3. The main uControl dashboard screen will be displayed



Supported Browsers

Recommended

These browsers have been tested and we recommend using:

  • Chrome version x and above
  • Safari version x and above
  • Firefox version x and above
  • Internet Explorer 10

Deprecated

These browsers are not supported:

  • Firefox version x and below
  • Internet explorer version 8 and below

 

User Security

The uControl administrator will have setup your account, provided you with a password and set your access rights.

All the actions on the system are recorded against a user ID for audit tracking.

Changing your password

  1. Log in to uControl using your assigned username and password
  2. Click the User Icon on the top right hand side of the UI

  3. In the pop-up box enter your new password and click save

     

 

Logging Out

  1. To log out of uControl click the Logout button to the right of the User Icon on the top right of the UI.

  2. This will return you to the login screen.
