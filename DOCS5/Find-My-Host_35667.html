uControl Documentation 5.0 : Find My Host

Created by Steve Hicks on Jul 08, 2016

The Find My Host tool under uManage is used to identify where a host is located within your ADDM estate

Find My Host

  1. Navigate to uManage -> Tools -> Find My Host



  2. Search by either hostname or IP address
