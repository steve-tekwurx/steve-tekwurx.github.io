Setting Password / Security Policies

To configure the security policy for uControl users:

  1. Login to uControl as an administrator privileged user
  2. Navigate to uAdmin -> Manage UserÂ 
  3. Select the Security Policy button

  4. Configure the settings as required

Update System Params

To customise your uControl instance the following updates / changes should be made to the system params:

  1. Login as a administrator privileged account
  2. Navigate to uAdmin -> System -> System params
  3. Update the parameters to customise the uControl appliance.
ParameterDescription
fileDownloadDirThis is the directory that will be used by uControl on the local system to store files retrieved files from ADDM
uControlUrlThe URL of the uControl appliance
fromEmailThe email address that notifications and alerts will be received / sent from
companyNameThe name of the company that is using uControl
